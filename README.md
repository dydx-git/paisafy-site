# Paisafy companion site

The marketing + legal site for **Paisafy**, a privacy-first personal finance tracker by Doorway Labs.

Static HTML/CSS, served via GitHub Pages:

- `index.html` — landing page (features, screenshots)
- `faq.html` — frequently asked questions
- `privacy.html` — privacy policy
- `assets/` — stylesheet and screenshots

No build step. Edit the HTML directly; changes go live on push.
